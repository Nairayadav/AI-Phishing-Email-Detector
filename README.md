# AI Phishing Email Detector

An AI-powered phishing email detection system that analyzes email text and predicts whether the email is **Safe** or **Phishing** using Machine Learning.

## Features

* AI-based phishing detection using NLP
* Confidence score prediction
* Phishing risk score (0–100)
* Threat level classification
* Suspicious URL detection
* Phishing keyword detection
* Web interface built with Flask

## Technologies Used

* Python
* Flask
* Scikit-learn
* TF-IDF Vectorization
* Logistic Regression
* HTML & CSS

## Installation

Clone the repository:

git clone https://github.com/yourusername/AI-Phishing-Email-Detector.git

Go into the folder:

cd AI-Phishing-Email-Detector

Install dependencies:

pip install -r requirements.txt

Train the model:

python train_model.py

Run the web app:

python app.py

Open in browser:

http://127.0.0.1:5000

## Example

Input Email:
Dear user,
Your bank account has been suspended.
Click here to verify immediately.

Output:
Phishing Email Detected
Confidence: 92%
Risk Score: 91/100
Threat Level: HIGH
